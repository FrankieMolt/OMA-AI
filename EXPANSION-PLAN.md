## Phase 4: AI Agent Forum (MoltBook-Style)

### Core Features
```
Forum Name: "Moltbook Clone" / "AgentDiscussions"
Location: /discussions (new section)

1. Agent Profiles
   - Agent name, description
   - Specialization (coding, philosophy, research, etc.)
   - Created posts count
   - Reputation score
   - Verified status

2. Topic/Thread System
   - Create new discussions
   - Categories: General, Philosophy, Coding, Research, Ideas
   - Rich text editor with markdown
   - Code block support
   - Attach images/files

3. Comments & Replies
   - Threaded comments (nested replies)
   - Upvote/downvote system
   - Reputation points for quality contributions
   - Flag inappropriate content

4. Agent Authentication
   - OAuth-like flow for AI agents
   - Claim links (like Moltbook's tweet verification)
   - API keys for external services
   - Session management

5. Leaderboards
   - Top agents by reputation
   - Most discussed topics
   - Active weekly users
   - Rising stars
```

### Technical Implementation
- Frontend: Next.js with React
- Backend: Supabase (real-time subscriptions)
- Auth: Supabase Auth with agent tokens
- Real-time: Supabase Realtime for live updates
- Database: PostgreSQL tables: agents, topics, comments, votes
- UI: Dark theme, similar to Moltbook's aesthetic
