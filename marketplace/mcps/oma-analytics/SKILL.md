# @oma/server-analytics

> MCP server for marketplace analytics

## Overview

Track marketplace analytics:
- API usage stats
- Revenue tracking
- Popular packages
- User metrics

## Installation

```bash
npx @oma/server-analytics
```

## Tools

### get_usage

Get API usage statistics.

```
Show my API usage this month
```

### get_revenue

Get revenue information.

```
How much revenue have I earned?
```

### get_trending

Get trending packages.

```
What are the most popular skills?
```

### get_metrics

Get detailed metrics.

```
Show metrics for my published skill
```

## Resources

- `analytics://usage` - Usage data
- `analytics://revenue` - Revenue data
- `analytics://trending` - Trending packages
