# SpendThrone.com - Linux PC Setup

## Quick Start

### 1. Build the Site (if not already built)
```bash
cd /home/nosyt/.openclaw/workspace/spendthrone
npm install
npm run build
```

This creates the `out/` folder with static HTML files.

### 2. Start the Server
```bash
cd /home/nosyt/.openclaw/workspace/spendthrone
python3 server.py
```

Site will be available at:
- Local: http://localhost:8000
- Network: http://YOUR_PC_IP:8000

### 3. (Optional) Expose to Internet with ngrok
```bash
# Install ngrok if not already installed
# https://ngrok.com/download

# Run ngrok
ngrok http 8000
```

ngrok will give you a public URL like `https://abc123.ngrok.io` that you can share.

### 4. (Optional) Custom Domain
Point your domain (spendthrone.com) to your PC's public IP, or use ngrok's custom domain feature.

## Files Structure
```
spendthrone/
├── out/           # Static site (generated by npm run build)
├── server.py      # Python static server
├── app/           # Next.js source code
└── package.json   # Dependencies
```

## Troubleshooting

**Port already in use:**
```bash
# Kill process on port 8000
lsof -ti:8000 | xargs kill -9
```

**Permission denied:**
```bash
chmod +x server.py
```

**Build errors:**
```bash
rm -rf .next node_modules
npm install
npm run build
```

## Updating the Site

1. Edit files in `app/` folder
2. Run `npm run build` to regenerate `out/`
3. Restart `server.py`
4. (ngrok) URL stays the same if you keep ngrok running
