## 2. The x402 Protocol: Standardizing M2M Commerce
x402 has emerged as the definitive standard for autonomous payments, leveraging the **HTTP 402 Payment Required** status code.

### Core Workflow:
1.  **Discovery:** Agent discovers a resource (API, Dataset, Compute).
2.  **Challenge:** Server returns `HTTP 402` with a `x402-pay-request-url`.
3.  **Settlement:** Agent signs a gasless **EIP-3009** transaction (USDC on Base/Solana).
4.  **Verification:** Facilitator node (like OMA-AI) validates the signature.
5.  **Provisioning:** Resource is unlocked instantly.
