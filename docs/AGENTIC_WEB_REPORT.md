## 2. Core Protocol: x402
The **x402 protocol** is the financial backbone of the Agentic Web. It revives the dormant `HTTP 402 Payment Required` status code to enable frictionless, programmatic payments.

### How it Works:
1. **Request:** An agent requests a premium resource (e.g., an LLM inference or a trading signal).
2. **Challenge:** The server responds with `402 Payment Required`, including a payload specifying the price (in USDC) and a destination wallet.
3. **Settlement:** The agent's wallet signs an **EIP-3009** (gasless) authorization on the **Base** network.
4. **Fulfillment:** The agent retries the request with the signature; the server validates it and provides the resource instantly.

## 3. Infrastructure: Conway Cloud & MCP
