# OMA-AI 2026 Roadmap

OMA-AI is building the infrastructure for **autonomous AI agents** that have identity, can pay for services, and travel anywhere. The MCP marketplace is live now at [oma-ai.com/mcps](https://oma-ai.com/mcps).

## Current Status (Now)

### ✅ Completed
- Platform infrastructure (Supabase, Next.js 15, Vercel)
- MCP Marketplace with 21 MCPs (free + x402 paid)
- x402 gasless payments on Base (USDC)
- Multi-chain wallet support (MetaMask/Base)
- Cross-Platform Plugins — Telegram, Discord, Slack, WhatsApp, webhooks
- Publisher Dashboard — earnings, analytics, payout management
- Developer documentation and API

### 🔄 In Progress
- **x402 Payment Signing** — HMAC-SHA256 signatures for gasless micro-payments
- **Treasury Management** — Multi-sig treasury with automatic revenue distribution
- **Agent DID + KYA** — Decentralized identity with Know Your Agent standards
- **Portable Agents** — Package agent with identity, wallet, and skills
- **Base Wallet + AAWP** — Programmatic wallet creation on Base
- **Micro-Payment Channels** — Batched transactions for high-frequency API calls
- **Programmable Guardrails** — Spending limits, contract allowlists, multi-party approvals
