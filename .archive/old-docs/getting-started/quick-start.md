# Quick Start Guide

Get started with OMA-AI in 3 simple steps.

## 1. Create an Account

\`\`\`bash
cd OMA-AI
npm install
npm run dev
\`\`\`

## 2. Deploy Your First Agent

Visit the dashboard and deploy your first autonomous agent.

\`\`\`bash
vercel --prod --yes
\`\`\`
