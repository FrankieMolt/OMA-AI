# Intelligence Storage System

A comprehensive system for storing, analyzing, and monetizing intelligence gathered from #secondbrain content.

## Architecture

- **Core**: Node.js + Express
- **Database**: SQLite + FTS5 (Full Text Search)
- **Integration**: Connects to Ultimate Ecosystem APIs
- **Monetization**: x402 payment gating for premium insights

## Features

1. **Content Ingestion**: Webhook/API to receive #secondbrain links/text
2. **Analysis**: Auto-categorization (Insight, Opportunity, Research)
3. **Storage**: Structured data with searchable metadata
4. **Access**: API endpoints for ecosystem consumption
5. **UI**: Dashboard integrated into Ultimate Ecosystem

## Setup

```bash
npm install
node server.js
```
