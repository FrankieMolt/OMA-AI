API Documentation

Complete reference for OMA-AI's API endpoints, authentication, and integration.

Introduction

OMA-AI provides a unified API for AI agents and developers to access:

  • ✅ Crypto prices (CoinGecko)
  • ✅ Weather forecasts
  • ✅ Web search
  • ✅ AI text generation
  • ✅ Text embeddings
  • ✅ Compute deployment

Authentication

API Keys

Include your API key in the Authorization header:

curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://oma-ai.com/api/price

x402 Payments

For paid endpoints, include the payment header:

curl -H "X-Payment: BASE64_ENCODED_PAYMENT" \
  https://oma-ai.com/api/premium-price
GET

/api/price

Get cryptocurrency prices for BTC, ETH, and SOL.

Request

GET https://oma-ai.com/api/price

Response

{
  "success": true,
  "data": {
    "btc": { "price": 66159, "change_24h": 3.15, "symbol": "BTC" },
    "eth": { "price": 1930, "change_24h": 4.78, "symbol": "ETH" },
    "sol": { "price": 82, "change_24h": 5.93, "symbol": "SOL" }
  },
  "source": "coingecko",
  "timestamp": 1708000000000
}
Free Rate limit: 30/min
GET

/api/weather

Get weather forecasts for any city.

Parameters

city City name (default: New York)

Example

GET https://oma-ai.com/api/weather?city=London
$0.002/call

SDK Examples

JavaScript / TypeScript

import { OMAClient } from '@oma-ai/sdk';

const client = new OMAClient();

// Get crypto prices
const prices = await client.prices.get();
console.log(prices.btc.price);

// Get weather
const weather = await client.weather.get('San Francisco');

// AI generation
const result = await client.ai.generate('Write a poem');
