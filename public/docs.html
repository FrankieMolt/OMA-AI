API Endpoints

OMA-AI provides 16+ production-ready APIs for AI agents. All endpoints support instant x402 micropayments on Base and Solana networks. No API keys required for the free tier.

GET /api/prices $0.001/call

Get real-time cryptocurrency prices from CoinGecko. Supports 100+ cryptocurrencies including BTC, ETH, SOL, and more. Updates every 30 seconds.

curl https://oma-ai.com/api/prices

# Response
{
  "success": true,
  "data": {
    "bitcoin": { "price": 67000, "change_24h": 2.5 },
    "ethereum": { "price": 3500, "change_24h": 1.2 }
  }
}

Authentication & Payments

x402 Micropayment Protocol

OMA-AI uses the x402 payment protocol for instant, trustless micropayments. For paid APIs (Pro tier), include the X-Payment header with your signed transaction.

# Free tier - no authentication needed
curl https://oma-ai.com/api/prices

# Pro tier - include x402 payment
curl -H "X-Payment: " \
     https://oma-ai.com/api/search?q=bitcoin

Payments settle instantly on-chain. You only pay for what you use. No subscriptions, no monthly fees.
