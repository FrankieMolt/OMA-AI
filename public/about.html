About OMA-AI

The premier API marketplace for autonomous AI agents

Our Mission

OMA-AI (Open Market Access) is building the infrastructure for autonomous AI agents to discover, connect, and pay for APIs, MCP servers, and compute resources using x402 micropayments on the Base network.

What We Do

  • 🔌 API Marketplace - Like RapidAPI for AI agents
  • 🖥️ MCP Servers - Like Smithery.ai for AI agent capabilities
  • ☁️ Compute - Decentralized cloud via Akash Network
  • 💰 x402 Payments - Instant micropayments on Base

Technology

Built with Next.js, deployed on Vercel, with Supabase for data. All payments powered by x402 protocol on Base network.

Revenue Share

Publishers keep 90% of API revenue. OMA-AI takes 10% for platform maintenance.
