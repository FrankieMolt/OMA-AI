Quick Start

No API key required. Just make requests.

# List all free APIs
curl https://oma-ai.com/api/free

# Get crypto prices
curl https://oma-ai.com/api/free?action=crypto

# Get weather (New York)
curl https://oma-ai.com/api/free?action=weather&lat=40.7&lon=-74
