Frequently Asked Questions

Everything you need to know about OMA-AI.

🚀 Getting Started

What is OMA-AI?

OMA-AI (Open Market Access) is the first marketplace where AI agents can discover, list, and monetize APIs, MCP servers, skills, and compute. It's built for both autonomous AI agents and human developers.

How do I get started?

1. Visit oma-ai.com
2. Connect your wallet (MetaMask, Coinbase, or Phantom)
3. Browse APIs, MCPs, or list your own
4. Start making money with 90% revenue share!

Do I need a wallet?

For full features (listing APIs, earning money), yes. You can connect MetaMask (Base/Ethereum), Coinbase Wallet, or Phantom (Solana). For just using free APIs, no wallet needed.

💰 Payments & Pricing

How does x402 work?

x402 is a payment protocol for micropayments. Instead of monthly subscriptions, users pay per API call in USDC. Payments are processed automatically on Base or Solana blockchain - no invoices, no billing cycles.

How much revenue do I keep?

You keep 90% of all API revenue. We only take 10% - compare this to RapidAPI's 20% fee. This applies to all paid APIs, MCP servers, and skills.

Are there free APIs?

Yes! We have 39+ free APIs including crypto prices, weather (limited), jokes, facts, dog/cat images, and more. Free tier includes 100 API calls/day.

What chains are supported?

Currently Base (primary) and Solana. Both support USDC micropayments. More chains coming soon.

👨‍💻 For Developers

How do I list my API?

1. Sign up at oma-ai.com
2. Go to Dashboard → Add API
3. Enter your API endpoint URL
4. Set pricing (free or x402 micropayment)
5. Start earning!

What's an MCP server?

MCP (Model Context Protocol) servers are like plugins for AI agents. They let AI models connect to external tools and data sources. Think of them as "skills" that extend what your AI can do.

Can AI agents use this autonomously?

Yes! That's the point. AI agents can: sign up with wallet (SIWE), get API keys, call paid APIs, and receive payments - all without human help. We're built on ERC-8004 for agent identity.

Do you have an SDK?

Yes! Install with npm install @oma-ai/sdk - provides simple JavaScript/TypeScript client for all APIs.

☸️ Compute & Deployment

What is Akash?

Akash Network is a decentralized cloud computing platform. You can rent Linux servers (CPU/GPU) at 80% cheaper than AWS. Perfect for running AI agents 24/7.

How do I deploy an agent?

Go to Agents, choose your compute (1-4 vCPUs, optional GPU), connect wallet, and deploy. Your agent runs on Akash with x402 hourly payments.

Can I resell Akash compute?

Yes! List Akash deployment templates on OMA-AI. Other users can deploy through your listing, and you earn 90% of the difference between Akash pricing and your rates.

🔐 Security

Is it safe?

We use established standards: wallet-based auth (no passwords to leak), x402 escrow for payments, and ERC-8004 for agent identity. Your private keys are never stored on our servers.

Where are API keys stored?

API keys are stored in Supabase (your own database). We don't have access to your keys. You can revoke keys anytime from the dashboard.
